## Run Locally

```bash
# Clone the repo
git clone https://github.com/mominrazashahid/mominrazashahid.github.io.git
cd mominrazashahid.github.io

# Option 1: Open index.html in a browser

# Option 2: Serve with a local server (e.g. Python)
python3 -m http.server 8000
# Then visit http://localhost:8000
```
